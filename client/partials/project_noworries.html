Project: "No Worries"

No Worries had to be "quick and painless". That was the driving idea behind this Rails app. The user had to be able to set up ‘tasks’ with minimal effort as possible.

Concept

"Going on a hike alone? Set a 'task', completion time, and a emergency contact. If you are unable to check in before the completion time, we'll send your contact a text as well as an email which contains a map that contains your starting point."

Issues

Suprisingly there were no major issues while creating this Rails app. The Rails community is very robust and supportive, most of the issues I solved by researching online.

Final

This project was made with the intention of helping people. I wondered what would happen if someone went missing on their trip? Who would know? That question spawned this project and I am ecstatic how it came out.
