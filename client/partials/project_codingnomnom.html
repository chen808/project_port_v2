Project: "Coding Nom Nom"

Initially, "Coding Nom Nom" was intended to be a place where programmers can look up a particular issue and find the steps on how to solve it. We wanted this website to "trim the fat" and get straight to the answer so programmers can get back to work.

Concept

We wanted to build a web app where programmers is able to find the solution quickly, and proceed with their project. We also wanted a web app where users are able to create their recipes on how they solved a particular issue.

Issues

Some of the issues we had was storing data in Postgres. We initiallly gave the user twenty lines to post their recipes, which presented an issue as it slowed down the process. We solved this by creating another table and split the recipe intake. That improved the program dramatically!

Final

This project was built because we wanted a web app where we can just find the solution quickly. Again, we believe in "strength through struggle" but sometimes a programmer doesn't have the luxury of time. We also encouraged use to write concise steps so anybody is able to understand from novice to experienced programmers alike.
